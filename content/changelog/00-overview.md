# Claude Code 更新日志（中文）

> 版本：2026-05-29  
> 来源：官方 [Changelog](https://code.claude.com/docs/en/changelog) 与 [anthropics/claude-code CHANGELOG.md](https://github.com/anthropics/claude-code/blob/main/CHANGELOG.md)  
> 性质：官方更新日志的中文镜像，由社区维护，按需随官方更新同步。

这是官方 Claude Code 更新日志的**中文全量镜像**，覆盖从 `0.2.21` 到最新版本的全部发布记录。它的作用是让中文读者不必逐条啃英文 changelog 就能跟上版本变化。

## 阅读说明

- **分组方式**：官方原文是扁平的 bullet 列表，本中文版把每个版本的改动归类为「新功能 / 改进 / 修复 / 安全 / 其他」，更易扫读。只有该版本确实包含的分组才会出现。
- **保留英文的内容**：命令（如 `/model`、`claude agents`）、参数（如 `--worktree`）、环境变量（如 `CLAUDE_CODE_USE_POWERSHELL_TOOL`）、设置键、文件名、模型 ID 和代码片段一律保留原样，只翻译说明性文字。平台标签如 `[VSCode]` 也保留。
- **日期**：GitHub 上的原始 CHANGELOG.md **不含日期**，日期只在官方文档站的最近若干版本可见。因此本镜像只对近期版本标注日期，更早版本仅保留版本号。
- **版本号不连续**：官方会跳过部分内部构建号（如 2.1.155、2.1.151、2.1.146 缺失），属正常现象，不是遗漏。

## 维护说明

- 官方每次发新版后，在本仓库提出更新请求，即把新版本**追加到对应系列文件的顶部**（保持新版在上）。
- 功能事实以官方 changelog 为唯一基线；翻译只做语言转换和分组，不增删技术含义。
- 与主指南第十二章「最新功能与版本跟踪」配合：那一章是**精选解读**，本日志是**全量镜像**。
