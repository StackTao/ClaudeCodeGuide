核验要点（均以 §164 官方 changelog 为准）：

- **Opus 4.8**：v2.1.154（2026-05-28）发布，默认 high effort，`/effort xhigh` 应对最难任务；模型 ID `claude-opus-4-8`。v2.1.156（2026-05-29）修复 4.8 下 thinking block 被改写导致的 API 报错。
- **Dynamic workflows**：v2.1.154 起 Claude 可在后台编排数十到上百个 agent，`/workflows` 查看。
- **`/model` 默认行为反转**：v2.1.153 起 `/model` 默认把所选模型保存为新会话默认，按 `s` 仅本次；旧的 `d`（`modelPicker:setAsDefault`）键位改名为 `modelPicker:thisSessionOnly`。本知识库原 12.1 中“默认 session-only、按 d 设默认”的说法已据此更正。
- **`/simplify` ≠ `/code-review`**：v2.1.147 曾把 `/simplify` 并入 `/code-review`，v2.1.152 让 `/simplify` 等于 `/code-review --fix`，v2.1.154 又让 `/simplify` 独立为清理向（复用/简化/效率/altitude）。当前是两个不同命令。
- **安全修复**：v2.1.149 修复多处 PowerShell 权限绕过（`cd..`/`cd\`/`cd~`/`X:`、过期 `PWD`/`OLDPWD`/`DIRSTACK`）与 worktree sandbox 写白名单过宽。
- **Auto mode**：v2.1.152 起不再需要事先 opt-in 同意；v2.1.154 起 AskUserQuestion 仅在确实无法自行决定时弹出。
