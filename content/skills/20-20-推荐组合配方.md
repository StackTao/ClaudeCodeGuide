## 20. 推荐组合配方

下面是按场景裁剪的组合，不是一次性安装清单。默认从官方能力开始，再按任务风险逐步加第三方插件。

### 20.1 稳健工程默认组合

- 官方 Commands / bundled skills。
- 语言 LSP。
- GitHub。
- `/code-review`。
- Code Simplifier（先在个人分支试用）。

适合：大多数严肃代码项目。

### 20.2 高风险 PR 组合

- Superpowers（复杂改动时启用）。
- `/code-review high` / `/security-review`。
- Codex plugin adversarial review。
- Trail of Bits security skills。
- GitHub + CI 日志。

适合：权限、支付、数据迁移、基础设施、安全敏感变更。

### 20.3 大型项目自动化组合

- GSD（隔离试用后启用）。
- Superpowers（先 plan / TDD / review 流程）。
- Git worktrees。
- `/goal`。
- LSP。
- focused-test-runner skill。

适合：多阶段迁移、长期 roadmap、上下文容易腐烂的大项目。

### 20.4 前端交付组合

- Superpowers。
- TypeScript LSP。
- Context7。
- Figma。
- Frontend Design。
- Browser/Playwright。
- Code Simplifier。

适合：真实产品 UI，不只是 demo。

### 20.5 调研与内容组合

- last30days（社区线索，隔离使用）。
- document-skills。
- Notion/Atlassian。
- Context7。
- docs-sync skill。

适合：行业调研、文档写作、知识库维护。
