## 24. 最终推荐路径

第一周：

```text
官方 Commands / bundled skills + 语言 LSP + GitHub
```

第二周：

```text
Superpowers（复杂任务）+ Code Simplifier（个人分支）+ Context7
```

第三周：

```text
Codex plugin（二次复核）+ document-skills + last30days（隔离调研）
```

大型项目再评估：

```text
GSD + Trail of Bits + 团队私有 plugin marketplace
```

不要一次装满。插件生态的正确用法不是“越多越强”，而是让每个插件都有明确职责、明确触发条件、明确安全边界。

团队落地时，把“默认启用”和“按需调用”分开：

- 默认启用：官方 LSP、GitHub、少量稳定 commands。
- 按需调用：Superpowers、GSD、Codex plugin、last30days、第三方大包。
- 高风险禁用或审批：未知 marketplace、会自动运行 shell 的 hooks、可访问生产数据的 MCP。
