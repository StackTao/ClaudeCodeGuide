## 附录 A：命令与能力速查

具体命令会随 Claude Code 版本变化，应以官方 CLI reference、Commands 页面和本机 `/help` 为准。详细教程、案例和 X/Twitter 实战线索见 `commands-and-cases.md`。

高频命令速查：

- 启动交互式会话：`claude`
- 单次脚本化查询：`claude -p "query"`
- 继续最近会话：`claude -c`
- 恢复指定会话：`claude -r "<session>"`
- 初始化项目说明：`/init`
- 编辑 memory：`/memory`
- 管理权限：`/permissions`
- 进入计划模式：`/plan`
- 设置完成条件并自动持续工作：`/goal`
- 清空上下文：`/clear`
- 压缩上下文：`/compact`
- 查看上下文占用：`/context`
- 查看成本/用量：`/usage`、`/cost`、`/stats`
- 管理 MCP：`claude mcp`、`/mcp`
- 管理 subagents：`/agents`、`claude agents`
- 审查改动：`/code-review`、`/simplify`、`/review`、`/security-review`
- 查看 diff：`/diff`
- 复制上一轮回复或代码块：`/copy`
- 运行和验证项目：`/run`、`/verify`（v2.1.145+）
- 调整推理努力：`/effort`（`high`/`xhigh`/`max`，Opus 4.8 默认 high）
- 后台多 agent 工作流：`/workflows`（v2.1.154+）
- 重扫 skill 目录：`/reload-skills`（v2.1.152+）
- 大规模并行迁移：`/batch`
- PR 自动修复：`/autofix-pr`
- 定期轮询：`/loop`
- 云端 routines：`/schedule`
- Claude API / SDK 迁移辅助：`/claude-api`
- 深度计划和审查：`/ultraplan`、`/ultrareview`
- GitHub App 接入：`/install-github-app`
- 语音输入：`/voice`
